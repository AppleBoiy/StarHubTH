*   **รันเกมได้ง่ายๆ**: เลือกรันเกมได้ทั้งโหมดปกติ (Vanilla) และโหมดผ่าน SMAPI สำหรับเล่นม็อด
*   **จัดการส่วนเสริม (Mods Manager)**:
    *   เปิด/ปิด ม็อดต่างๆ ได้อย่างง่ายดาย
    *   **ใหม่!** ติดตั้งม็อดอัตโนมัติจากการลากไฟล์ `.zip` หรือโฟลเดอร์มาวาง
    *   **ใหม่!** ระบบคัดกรองม็อด (Filter & Tags) ตามหมวดหมู่และสถานะ
    *   **ใหม่!** ระบบ Backup Mods สรุปรวมโฟลเดอร์ม็อดเป็นไฟล์ `.zip` ในคลิกเดียว
*   **แก้ไขตั้งค่าม็อด (Mod Config Editor)**: แก้ไขไฟล์ตั้งค่าของม็อดได้จากในแอปโดยตรง รองรับรูปแบบเมนูซ้อนทับ (Nested Menu) เหมือนในเกมเป๊ะๆ
*   **เชื่อมต่อ Nexus Mods**: ดูรายละเอียด อัปเดตข้อมูล และดาวน์โหลดม็อดจาก Nexus Mods ได้โดยตรงจากในแอป
*   **มอดแพ็ค (Mod Packs)**:
    *   **ใหม่!** นำเข้าคอลเลกชัน Nexus ผ่านลิงก์ `nxm://` โดยตรง
    *   **ใหม่!** แสดงรายละเอียดคอลเลกชันแบบครบถ้วน: รูปหน้าปก, ชื่อผู้สร้าง, จำนวนดาวน์โหลด, รีวิชัน, เวอร์ชันเกม
    *   **ใหม่!** ตรวจสอบสถานะม็อดแต่ละตัวในคอลเลกชันได้ทันที (ติดตั้งแล้ว / ปิดใช้งาน / ขาดหาย)
*   **โปรไฟล์ม็อด (Mod Profiles)**: จัดกลุ่มม็อดเป็นหลายโปรไฟล์และสลับใช้งานได้ทันทีในคลิกเดียว
*   **มือแปลไทย (Thai Translation Hub)**: ศูนย์รวมม็อดแปลภาษาไทยทั้งหมด — ดู, ตรวจสอบ, ดาวน์โหลด ในที่เดียว
*   **จัดการเซฟเกม (Save Manager)**:
    *   ดูรายละเอียดเซฟเกมทั้งหมด (จำนวนเงิน, เวลาในเกม, ฤดูกาล, รูปแบบฟาร์ม)
    *   ทำสำเนา (Duplicate) หรือลบเซฟเกม
    *   แก้ไขเงินและสถานะต่างๆ ของตัวละครเบื้องต้น
*   **บันทึกนักพัฒนา (Developer Logs)**: ติดตามการทำงานของ SMAPI ได้แบบเรียลไทม์ภายในแอป
*   **ระบบแปลภาษาอัจฉริยะ**: ติดตามและตรวจสอบสถานะม็อดแปลภาษาไทย
*   **รองรับ 2 ภาษา (Bilingual Support)**: สลับภาษาในแอปได้ทันทีระหว่างภาษาอังกฤษ (English) และภาษาไทย (Thai)
*   **UI สไตล์ Native macOS**: หน้าตาแอปพลิเคชันที่สวยงาม ใช้งานง่าย ออกแบบมาให้กลมกลืนกับระบบ macOS อย่างสมบูรณ์แบบ

1. **ดาวน์โหลดแอปพลิเคชัน**: โหลดไฟล์เวอร์ชันล่าสุดจากหน้า [Releases](../../releases)
2. **เปิดใช้งาน**: แตกไฟล์แล้วลาก `StarHubTH.app` ไปไว้ที่โฟลเดอร์ Applications แล้วดับเบิลคลิกเพื่อเปิดใช้งาน
3. **กำหนดโฟลเดอร์เกม**: ในครั้งแรกที่เปิด โปรแกรมจะค้นหาโฟลเดอร์เกมของ Steam อัตโนมัติ หากไม่พบ คุณสามารถเลือกโฟลเดอร์ตัวเกม (เช่น `/Applications/Stardew Valley.app/Contents/MacOS`) ได้ด้วยตัวเอง
4. **พร้อมลุย!**: จัดการม็อดหรือเซฟเกม แล้วกด **"เข้าสู่เกม"** จากแถบเมนูด้านซ้ายได้เลย!

แอปพลิเคชันนี้เขียนขึ้นด้วย **Swift** และ **SwiftUI** ผ่านโครงสร้างของแอปพลิเคชัน macOS แท้ๆ

### ความต้องการของระบบ (Requirements)
*   macOS 13.0 (Ventura) หรือใหม่กว่า
*   Xcode 15.0 หรือใหม่กว่า (สำหรับการคอมไพล์ซอร์สโค้ด)

### วิธีการรันโปรเจกต์
คุณสามารถเปิดโปรเจกต์ผ่าน Xcode หรือใช้สคริปต์คอมไพล์ผ่าน Terminal:
```bash
python3 build_app.py
open StarHubTH.app
```

### การแพ็คแอปพลิเคชัน (Release)
หากต้องการบีบอัดแอปพลิเคชัน (.app) เป็นไฟล์ `.zip` สำหรับนำไปแจกจ่าย สามารถรันคำสั่ง:
```bash
python3 release.py
```
ไฟล์ Release จะถูกบันทึกไว้ในโฟลเดอร์ `bundles/` ครับ

### การรันเทส (Tests)
```bash
python3 run_tests.py
```

### เอกสารสำหรับนักพัฒนา (Developer Docs)
หากคุณจะเขียนหรือแก้ไขโค้ด Swift ในโปรเจกต์นี้ กรุณาอ่านเอกสารเหล่านี้ก่อน:

*   [**docs/SWIFT_STANDARDS.md**](docs/SWIFT_STANDARDS.md) — มาตรฐานการเขียนโค้ด Swift ของโปรเจกต์ (อ้างอิงจาก Swift API Design Guidelines ของ Apple) พร้อมตัวอย่างก่อน/หลังจากโค้ดจริงในโปรเจกต์ **บังคับใช้กับโค้ดใหม่ทุกบรรทัด**
*   [**docs/PROJECT_STRUCTURE.md**](docs/PROJECT_STRUCTURE.md) — โครงสร้างโฟลเดอร์ กฎการ import ของแต่ละชั้น และแผนที่ไฟล์ทั้งหมด ดูก่อนสร้างไฟล์ใหม่ทุกครั้ง
*   [**docs/REFACTOR_PLAN.md**](docs/REFACTOR_PLAN.md) — แผนการปรับโครงสร้างโค้ดแบบเป็นเฟส สำหรับการแก้โค้ดเดิม
